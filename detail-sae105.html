Description du projet

🐍 Script 1 — SelectDir.py

  • Ouvre un sélecteur de répertoire (PyQt5)
  • Utilise QFileDialog.getExistingDirectory()
  • Retourne le chemin au script PowerShell

🔍 Script 2 — AnalyseDisk.py

  • Parcours récursif avec pathlib rglob()
  • Liste [chemin, taille_octets] par fichier
  • Tri décroissant avec lambda
  • Filtre : seuil 1 MiB + max 100 fichiers
  • Export JSON (resultats.json)

📊 Script 3 — MainGraphique.py

  • Lit le fichier JSON
  • Génère 100 couleurs aléatoires (QColor)
  • Camembert PyQt5 + QChart (QPieSeries)
  • Légendes avec cases à cocher (25/page)
  • Génère delete_files.ps1 à la demande

💻 Run_SAE.ps1

  • Orchestre les 3 scripts Python
  • Valide le répertoire (Test-Path)
  • Passe le chemin en argument
  • Lance MainGraphique.py en dernier
  • Exécute delete_files.ps1 si généré
