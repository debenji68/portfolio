Projet NSI — Générateur de mots de passe

🔐 Application Python avec interface graphique

Développement d'une application complète pour générer des mots de passe robustes conformes aux recommandations de l'ANSSI.

  • Interface graphique Tkinter — paramétrage longueur, types de caractères
  • Options : majuscules, minuscules, chiffres, caractères spéciaux, longueur personnalisée
  • Copie automatique dans le presse-papiers (module pyperclip)
  • Indicateur de robustesse du mot de passe généré
  • Respect des bonnes pratiques de sécurité informatique
