The tool is orchestrated by a PowerShell script that chains the 3 Python scripts. Try the interactive demo below, based on my real files!

Project description

🐍 Script 1 — SelectDir.py

  • Opens a directory selector (PyQt5)
  • Utilise QFileDialog.getExistingDirectory()
  • Returns the path to the PowerShell script

🔍 Script 2 — AnalyseDisk.py

  • Recursive traversal with pathlib rglob()
  • List [path, size_bytes] per file
  • Descending sort with lambda
  • Filter: 1 MiB threshold + max 100 files
  • JSON export (resultats.json)

📊 Script 3 — MainGraphique.py

  • Reads the JSON file
  • Generates 100 random colours (QColor)
  • PyQt5 pie chart + QChart (QPieSeries)
  • Checkbox legends (25/page)
  • Generates delete_files.ps1 on demand

💻 Run_SAE.ps1

  • Orchestrates the 3 Python scripts
  • Validates the directory (Test-Path)
  • Passes the path as argument
  • Runs MainGraphique.py last
  • Runs delete_files.ps1 if generated
