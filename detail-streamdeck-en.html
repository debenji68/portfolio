DIY Stream Deck

Arduino Pro Micro · Key modules · 3D Printing · Soldering

HID keyboard emulation 100% homemade Customisable shortcuts

Building a programmable macro keypad similar to a Stream Deck. The Arduino Pro Micro is connected to mechanical key switches. The device emulates a USB keyboard (HID protocol) and allows launching custom keyboard shortcuts on PC: playing music, opening applications, triggering macros... The case is entirely modelled and 3D-printed. Components are hand-soldered.

Features

⌨️ PC control

  • USB keyboard emulation (HID)
  • Fully customisable keyboard shortcuts
  • Application launching
  • Music / volume control
  • Multi-key macros

🔌 Electronics

  • Arduino Pro Micro (ATmega32U4)
  • Cherry MX mechanical key switches
  • Direct USB power
  • Matrix wiring of keys
  • Manual soldering

💻 Arduino programming

  • Arduino C++ code
  • HID-Project library
  • Debounce management
  • Key definitions in array
  • Upload via Arduino IDE

🖨️ Manufacturing

  • Custom enclosure modelling
  • FDM printing (PLA)
  • Notches for key modules
  • Screwable lid
  • Full assembly

Tech stack

Arduino Pro MicroATmega32U4 USB HIDC++ Arduino HID-ProjectKey modules Soldering3D Modelling 3D PrintingArduino IDE

What I learned

  • Programming an Arduino microcontroller with HID emulation
  • USB HID protocol — how a keyboard device communicates with the PC
  • Debounce and key state management in Arduino code
  • Matrix wiring of mechanical keys with soldering
  • Designing a functional 3D enclosure with manufacturing tolerances
