BUT Networks & Telecommunications 2025–2028

IUT of Colmar — University of Haute-Alsace

// National Bachelor Bac+3 · Cybersecurity & Networks

The University Bachelor of Technology in Networks & Telecommunications is a 3-year program (180 ECTS) at the IUT of Colmar. It prepares students for careers in network administration, cybersecurity, embedded systems and software development. The program combines theoretical courses, practical network lab work and professional project SAÉ.

IUT of Colmar website

Modules — Semesters 1 & 2 (Year 1)

Introduction to networksNetwork principles & architecture LAN & active equipmentElectronic systems fundamentals Transmission mediaDigital system architecture Programming fundamentalsOperating systems basics Introduction to web technologiesTechnical English 1 Personal & professional projectSignal mathematics

Key content areas

  • Network administration — Advanced Cisco IOS configuration: VLANs, trunks, inter-VLAN routing, OSPF, ACL.
  • Cybersecurity — Firewalls (iptables, pfSense), intrusion detection (IDS/IPS), VPN, Wireshark frame analysis.
  • Operating systems — Linux administration (Debian, Ubuntu Server), Bash scripting, systemd, Active Directory on Windows Server.
  • Development — HTML5/CSS3, MySQL, React, full web applications. Python networking basics.
  • Virtualisation — VMware ESXi, VirtualBox, Docker and Docker Compose.
