UHA Competency Framework

3 Learning Units · Critical Learning Outcomes · Assessment Criteria

The BUT R&T programme at IUT Colmar (Université de Haute-Alsace) is built around 3 core competencies, each corresponding to a Learning Unit (UE). Each UE groups Critical Learning Outcomes (AC) — the concrete skills to master — and Assessment Criteria (CE) — the conditions under which these competencies are evaluated. This page presents the official BUT R&T framework as applied in my degree programme.
🌐
UE1 — Competency 1

Administer Networks and the Internet

Design, configure, deploy and maintain network infrastructures and Internet services. This competency covers all switching and routing equipment, fundamental network protocols, access security, as well as fault diagnosis and troubleshooting. It requires solid knowledge of electricity, transmission media and local network protocols.

Assessment Criteria (CE)

  • CE1.01 By choosing appropriate network solutions and technologies
  • CE1.02 By respecting the fundamental principles of IT security
  • CE1.03 By using a rigorous approach to troubleshooting
  • CE1.04 By respecting business rules
  • CE1.05 By maintaining technological watch

Critical Learning Outcomes (AC)

  • AC11.01 Master fundamental laws of electricity to work on network and telecom equipment
  • AC11.02 Understand transmission media and their characteristics
  • AC11.03 Configure and troubleshoot switching and routing equipment
  • AC11.04 Master protocols and technologies of local networks
  • AC11.05 Identify local network malfunctions

UE2 — Competency 2

Connect Companies and Users

Deploy and configure connectivity solutions to link companies, remote sites and end users to the Internet and Cloud services. This competency covers advanced networks (OSPF, BGP, VPN), secure wireless access, VoIP telephony and integration into heterogeneous environments, while respecting constraints of bandwidth, latency and availability.

Assessment Criteria (CE)

  • CE2.01 By choosing appropriate technical solutions
  • CE2.02 By respecting constraints of bandwidth, latency, availability and security
  • CE2.03 By integrating solutions into the existing environment

Critical Learning Outcomes (AC)

  • AC12.01 Deploy advanced network equipment
  • AC12.02 Configure advanced network services
  • AC12.03 Deploy secure wireless access
  • AC12.04 Set up a VPN infrastructure

UE3 — Competency 3

Create IT Tools and Applications

Design and develop software applications and tools suited to the needs of networks and telecommunications. This competency covers programming (Python, Django, C++), web architectures (client-server, APIs, databases), production deployment (Linux, Nginx, Gunicorn), application security, and the use of embedded systems. It values algorithmic rigour, collaborative work (Git) and autonomy in setting up services.

Assessment Criteria (CE)

  • CE3.01 By using appropriate development languages and environments
  • CE3.02 By respecting algorithmic and programming principles
  • CE3.03 By integrating security constraints

Critical Learning Outcomes (AC)

  • AC13.01 Use a computer system and its tools
  • AC13.02 Read, execute, correct and write a program
  • AC13.03 Translate an algorithm into a programming language
  • AC13.04 Know the architecture and technologies of a website
  • AC13.05 Set up IT tools and services
  • AC13.06 Use embedded computer systems
