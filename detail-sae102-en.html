Introduction to Computer Networks

Multi-VLAN network infrastructure · Cisco IOS · EVE-NG · Linux DHCP · Wireshark

Complete configuration of a network infrastructure for a branch of the fictional company "Rudy and Theodor Company" (RTC). The project involved setting up a network with 4 distinct VLANs (Management, Users, Servers, Administrator), implementing inter-VLAN routing, configuring a DHCP service on Linux, analysing 802.1Q frames with Wireshark and deploying a second switch with VTP.

Project description

🌐 VLAN infrastructure

  • 4 VLANs: Management, Users, Servers, Administrator
  • The 172.20.x.0/24 IP address range is divided into /29 subnets
  • Cisco Catalyst 2960 (switch) + Cisco 2800 (router)
  • EVE-NG lab — network simulation
  • Virtual PCs per VLAN for ping/traceroute tests

⚙️ Configured services

  • Inter-VLAN routing (Router-on-a-stick)
  • DHCP on switch then migrated to Linux Ubuntu
  • IP Helper-address for inter-VLAN DHCP relay
  • Wireshark capture of 802.1Q tagged frames
  • VTP v2 (Server/Client mode) between 2 switches

🔧 Cisco IOS commands

  • VLAN and trunk configuration (802.1Q)
  • Static routing and sub-interfaces
  • DHCP pools on switch and router
  • show running-config, show vlan, show ip route
  • SSH management to switch from Admin VPC
